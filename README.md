🌍** Empowering Lives — NGO Website**
Empowering Lives is a nonprofit organization dedicated to improving education, healthcare, and community development, fostering sustainable growth and positive change in society.

This website has been designed to showcase our mission, services, and community impact while encouraging visitors to get involved and make a difference.

**🚀 Features**
Responsive website design
About Us, Services, Gallery, and Contact sections
Donation information and social links
Simple and clean layout with accessibility in mind
Fully built using HTML, CSS, Bootstrap5 and JavaScript
**🛠️ Technologies Used**
**HTML5** – Structure and content
**CSS3** – Styling and layout
**Bootstrap5** - Responsive and Annimation
**JavaScript (ES6)** – Interactivity
Font Awesome – Icons
Responsive Design – Optimized for mobile and desktop
